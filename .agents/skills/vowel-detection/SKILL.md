# Vowel Detection

This skill is designed to analyze a given string of text and identify the presence of vowels. It counts the occurrences of each vowel (both uppercase and lowercase) and provides a summary of the counts.

## Process

1. Identify vowels:
  1. 'a' or 'A'
  2. 'e' or 'E'
  3. 'i' or 'I'
  4. 'o' or 'O'
  5. 'u' or 'U' 
2. Provides a count of each vowel found in the input text

## Usage

To use this skill, simply provide a string of text as input. The skill will analyze the text and return a count of each vowel present.

# Example

**Input:**

```
Hello World!
```

**Output:**

```
Vowel Counts:
a: 0
e: 1
i: 0
o: 2
u: 0
```
